# 🌍 shickotours - המלווה החכם לטיולים שלך

**shickotours** היא אפליקציית ווב מתקדמת המיועדת למטיילים המחפשים תכנון מסלול חכם, גמיש ומותאם אישית. האפליקציה מתבססת על לוגיקת ה-Gem המקורית והופכת אותה לממשק ויזואלי ידידותי עם דגש על נוחות שימוש בנייד ותמיכה מלאה בעברית.

---

## ✨ תכונות עיקריות (Features)

* **מפה אינטראקטיבית:** הצגת כל מסלול הטיול על גבי מפות גוגל עם סימון נקודות עניין (Markers).
* **מערכת Plan B חכמה:**
    * **זיהוי אוטומטי:** חיבור ל-API של מזג אוויר והפעלת תוכנית חלופית (מקומות מקורים) אם צפוי גשם (>40%).
    * **כפתור "פאניקה" ידני:** אפשרות למשתמש להפעיל את "מצב הגשם" בלחיצה אחת בכל רגע, ללא תלות בחיזוי.
* **אינטגרציה עם שירותי ניווט:** כפתורי "נווט" לכל עצירה המקשרים ישירות ל-Waze או ל-Google Maps.
* **ייצוא ל-PDF:** הורדת סיכום הטיול כקובץ PDF מעוצב הכולל את המפה הפעילה והמסלול שנבחר.
* **גילוי אטרקציות בסביבה:** הצגת המלצות נוספות בקרבת נקודות הציון בעזרת Google Places API.

---

## 🛠 טכנולוגיות (Tech Stack)

* **Frontend:** React (Vite) - מהיר, קל ורספונסיבי.
* **Styling:** Tailwind CSS עם תמיכה מלאה ב-RTL (יישור לימין).
* **APIs:** Google Maps JS API, Google Places API, OpenWeatherMap API.
* **Libraries:** `html2pdf.js` לייצוא מסמכים ו-`Lucide React` לאייקונים.

## 🚀 הגדרה והפעלה (Setup)

האפליקציה משתמשת במשתני סביבה (Environment Variables) כדי לשמור על אבטחת מפתחות ה-API.

### 1. הגדרת מפתחות API
צור קובץ בשם `.env` בתיקייה הראשית (או הגדר ב-GitHub Secrets) והוסף את השורות הבאות:

```env
VITE_GOOGLE_MAPS_API_KEY=your_google_maps_key_here
VITE_OPENWEATHER_API_KEY=your_openweather_key_here
