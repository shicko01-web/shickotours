

## בניית זרימת אונבורדינג — מסך פתיחה + טופס תכנון טיול

### מה הולך להיבנות

זרימה של 3 מסכים שמובילה את הגולש מהכניסה הראשונית עד למסלול מותאם אישית שנוצר ע"י ה-AI:

```text
┌─────────────┐     ┌──────────────┐     ┌─────────────────┐
│  1. פתיחה   │ ──▶ │ 2. טופס תכנון │ ──▶ │ 3. מסלול חכם    │
│  Hero +     │     │ אזור / משך /  │     │ (המסך הקיים     │
│  CTA "צור   │     │ סוג / חברה   │     │  היום, מוזן     │
│  טיול חדש"  │     │              │     │  מהטופס)        │
└─────────────┘     └──────────────┘     └─────────────────┘
```

### מסך 1 — Landing (`/`)

- Hero בעברית עם הלוגו `shickotours`, רקע גרדיאנט קיים
- כותרת: "תכננו טיול בארץ ב-30 שניות"
- תת-כותרת קצרה על מה שהאפליקציה עושה (AI + תחזית גשם)
- 3 כרטיסי "איך זה עובד" (תכנון / גילוי חכם / Plan B לגשם)
- CTA ראשי: **"צור טיול חדש"** → מנווט ל-`/plan`
- CTA משני: **"המשך טיול אחרון"** → מופיע רק אם יש טיול שמור ב-localStorage

### מסך 2 — טופס תכנון (`/plan`)

טופס בן 4-5 שדות, צעד-אחר-צעד או על מסך אחד (אבחר מה שיותר נקי במובייל):

| שדה | סוג | אפשרויות |
|------|-----|----------|
| אזור הטיול | Select / Combobox | גליל, גולן, ירושלים, נגב, מישור החוף, ים המלח, אילת... |
| תאריכי הטיול | Date range picker | התחלה + סיום |
| סוג הטיול | Chips (multi) | טבע, אוכל, היסטוריה, נופים, אתגרי, קליל |
| הרכב | Chips (single) | משפחה עם ילדים / זוג / חברים / יחיד |
| קצב | Slider | רגוע ↔ אינטנסיבי |

כפתור "צור לי מסלול" → קורא ל-Edge function שמייצר טיול מלא עם Gemini, שומר ב-localStorage ומנווט ל-`/trip`.

### מסך 3 — המסלול (`/trip`)

זה בעצם המסך הקיים היום (`Index.tsx`). רק נעביר אותו ל-route חדש ונדאג שיוזן מהנתונים שהמשתמש מילא.

### שינויים טכניים

1. **ראוטינג חדש** ב-`src/App.tsx`:
   - `/` → `Landing.tsx` (חדש)
   - `/plan` → `PlanTrip.tsx` (חדש)
   - `/trip` → התוכן הקיים של `Index.tsx`

2. **Edge function חדשה** `generate-trip`:
   - מקבלת את העדפות המשתמש
   - Gemini בונה מסלול מלא (stops + planB) על בסיס ההעדפות
   - Geocoding מאמת קואורדינטות לכל תחנה
   - מחזירה אובייקט `Trip` תקין

3. **עדכון `useTrip`** — לקבל גם `setTrip` מטופס, לא רק seed

4. **רכיבי UI חדשים**:
   - `Landing.tsx` — Hero + הסבר + CTA
   - `PlanTrip.tsx` — הטופס
   - `RegionSelect`, `TripStyleChips`, `PaceSlider` — שדות מותאמים

5. **שמירת state**: הטיול האחרון נשמר ב-localStorage (כבר קיים), כך שהחזרה לאתר תאפשר "המשך טיול אחרון"
